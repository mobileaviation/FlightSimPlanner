What is New

version 1.0.0.4
Direct access to weather and notams for every airport within an adjustable buffer around your current location or around your flightplan.
  • Notams from the FAA pilotweb site
  • Metars and Taf from aviationweather.gov
There is a very basic parsing of the position in the notam. When "psn" is in the notam text the software will try to parse this position and show a "Notam" maker on the map..
The Notams, Metars and Tafs are stored in the database so they can be retrieved when your device is offline..

Other changes in this new version:
  • Height added to in info-top control.
  • Height calculation bug in tracks fixed.
  • Layout enhanced for smaller devices.
  • Other bugfixes and enhancements
  • New database
